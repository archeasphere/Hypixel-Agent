This project is an agent that requires the following to function

A Hypixel API developer Key
- For this run /api in hypixel and follow the appropriate steps

A claude code subscription or a local agent to run the workflow
- This might work outside of just claude models but they are the easiest to integrate

A Hypixel skyblock profile with your APIs turned on
- If you don't know how to do this, search up SkyCrypt Hypixel skyblock, or there are plenty of youtube videos that explain the same thing

When running simply prompt the agent asking it for a profile review and it should work on its own from there

**Note: the agent isn't constantly being updated for the latest version of the game. If you need to update it, simply prompt the chat interface to update the workflow for the latest version of the game using the hypixel skyblock wiki.**
