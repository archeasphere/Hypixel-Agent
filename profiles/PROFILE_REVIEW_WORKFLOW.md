# Profile Review Workflow

## Step 1 — Required Questions Before Analysis

Always ask ALL of the following before starting any profile research:

1. **IGN** — What is the player's Minecraft username?
2. **Profile** — Which profile? (Strawberry, Mango, Coconut, etc.)
3. **Primary Goal** — What is the player focusing on?
   - More coins / Faster progression / Dungeons / Slayer / Mining / Farming / Combat / Ironman / Endgame optimization
4. **Specific Areas** — Any particular sections to deep-dive?
5. **Profile Type** — Regular, Ironman, Stranded, or Bingo?

---

## Step 2 — Research

- Load SkyCrypt: `https://sky.shiiyu.moe/stats/{IGN}/{Profile}`
- Verify profile exists before proceeding
- Review all sections: Skills, Accessories, Armor, Equipment, Pets, Bank/Purse, Collections, Minions, Dungeons, Slayer, Mining, Farming, Rift, Museum, HotM, Magical Power, Net Worth, Bestiary, Crimson Isle

---

## Step 3 — Report Structure

### Executive Summary
- Profile stage (Early / Mid / Late / End Game)
- Overall profile score
- Top 3 strengths
- Top 3 weaknesses
- Most impactful next upgrades

### Net Worth Analysis
- Total net worth
- Liquid coins (purse + bank + co-op purse if applicable)
- Asset distribution
- Coin efficiency assessment

> **Bank Balance Warning:** SkyCrypt frequently shows bank balance as 0 due to caching or API lag. Always ask the player to confirm their actual bank balance before calculating liquid coins or net worth.
> **Co-op Purse:** If the player is on a co-op profile, ask for the combined co-op purse balance — SkyCrypt may not accurately reflect contributions from all members.

### Skills Review
- Skill averages
- Lowest / highest skills
- Recommended priorities

### Magical Power Analysis
- Current MP
- Recommended target
- Missing accessories
- Cheapest upgrades

### Gear Review
- Armor / Weapon / Equipment assessment
- Missing upgrades

### Pet Analysis
- Best pets owned
- Missing progression pets
- Upgrade recommendations

### Dungeon Analysis
- Catacombs level, class levels, floor progression, gear readiness

### Slayer Analysis
- Slayer levels, missing unlocks, progression opportunities

### Mining Analysis
- HotM progression, powder, gear, income potential

### Farming Analysis
- Farming level, fortune, garden progression, profit potential

### Museum Analysis
- Completion percentage
- Special items donated (armor sets, weapons, rarities)
- Missing notable donations
- Armor sets completed vs. missing
- Museum score and tier
- Recommended donations (value vs. cost)
- Items worth donating for progression bonuses

### Money Making Assessment
- Best current methods
- Estimated coins/hour
- Recommended path + investment opportunities

### Progression Roadmap
- **Immediate** — Top 5 upgrades, cost, expected benefit
- **Short-Term** — 1–2 week targets
- **Mid-Term** — 1–2 month targets
- **Long-Term** — Major milestones

### Risk Assessment
- Common traps to avoid
- Wasteful upgrades
- Better alternatives

### Final Priority List
- Top 20 upgrades ranked by cost-to-benefit ratio

---

## Rules
- Ask all clarifying questions first — no exceptions
- Verify profile on SkyCrypt before reviewing
- Rank upgrades by value per coin spent
- Bullet points over paragraphs
- Beginner-friendly language
- Separate facts from opinions
- Include estimated costs whenever possible
- Highlight both strengths and weaknesses
- Focus on progression efficiency, not vanity
